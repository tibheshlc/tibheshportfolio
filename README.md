# Tibhesh Lamichhane — Digital Portfolio

A minimalist, high-performance digital portfolio showcasing the photographic and visual art work of **Tibhesh Lamichhane**, a photographer and visual artist based in Kathmandu, Nepal.

This project is designed with a premium, gallery-like aesthetic, emphasizing clean typography, smooth transitions, and high-quality imagery. It features a curated selection of commercial, editorial, and documentary projects.

## ✨ Features

- **Project Filtering**: Interactive filter system on the homepage to browse projects by category.
- **Dynamic Project Routing**: A dedicated project page template that loads content dynamically via URL parameters from a centralized data store.
- **Integrated Lightbox**: Full-screen image viewing with keyboard navigation and gesture support on project pages.
- **Responsive Design**: Fully optimized for all screen sizes, from mobile devices to high-resolution desktops.
- **Modern Aesthetics**: Utilizes `EB Garamond` for a timeless serif feel and `DM Sans` for modern readability.
- **Scroll Animations**: Uses `IntersectionObserver` for smooth, performant reveal animations as you scroll.
- **Micro-Animations**: Smooth entrance animations and hover effects to enhance user engagement.
- **Contact & About**: Dedicated sections for biographical information and professional inquiries.

## 🛠️ Built With

- **HTML5**: Semantic structure for SEO and accessibility.
- **CSS3**: Custom styling using CSS Variables, Flexbox, and CSS Grid for layout management.
- **Vanilla JavaScript**: Lightweight logic for filtering, mobile navigation, and dynamic content loading.
- **Google Fonts**: Integrated `EB Garamond` and `DM Sans`.

## 🚀 Getting Started

To view the portfolio locally:

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/tibheshlc/tibheshportfolio.git
    ```
2.  **Open in Browser**:
    Simply open `index.html` in any modern web browser.
3.  **Local Server (Optional but Recommended)**:
    For the best experience, especially if using certain browser-restricted features, run a local server:
    ```bash
    # Using Python
    python -m http.server 8000
    ```
